# STS 프로젝트

### 프로젝트 계획이유
--------------------
> 배워왔던 기술들을 가장 잘 표현할 수 있으면서도 실제 사용될만한 내용의 사이트를 만드는 것이 목적이었습니다. 때문에 복잡한 쿼리보단 최대한 단순하면서 재사용이 가능하고 다른 사람이 소스를 볼 때 기능파악 및 유지보수가 수월할 수 있도록 주석 및 소스코드를 정렬했습니다
> 
> 그래서 선정한 것은 영화 리뷰 커뮤니티 사이트 기본적인 CRUD 기능을 포함해 이미지 첨부를 이용한 갤러리 게시판, 스크랩, 추천수, 댓글, 회원 기능을 표현해 보았습니다.
> 
> 비회원인 경우 게시물 읽는 권한만 있고 회원은 리뷰 게시판이 사용가능하며 게시판 지기는 영화소개 게시판에 글을 업로드 할 수 있습니다.
>
